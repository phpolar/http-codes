# HTTP CODES

PHP class for HTTP Response Codes

## Installation

```shell
composer require phpolar/http-codes
```

## Usage

```php
// "Import" the class
use Phpolar\HttpCode\Code as HTTP;

// Use the code in your code.
echo HTTP::OK; // 200
```
